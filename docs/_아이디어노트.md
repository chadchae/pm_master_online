# Project Manager — 아이디어노트

Projects 폴더 내 모든 프로젝트, 문서, 노트, 이슈, 학습, 가이드라인을 웹에서 운영·관리하는 로컬 프로젝트 매니저.

---

## 구현 완료 기능 (v0.1.0 — 2026-03-19)

### 1. 인증
- 비밀번호 전용 (아이디 없음, 1인 사용)
- bcrypt 해시, 파일 기반 토큰 (서버 재시작 시 유지)
- 비밀번호 변경 (Settings)

### 2. 프로젝트 관리
- 칸반 보드 (Initiation → Development → Testing → Completed → Archived)
- 리스트 뷰 전환 (Kanban / List 토글)
- 드래그앤드롭으로 단계 이동 → 작업지시 모달 → 실제 폴더 이동
- 프로젝트 카드: label(정식이름), description, 유형 뱃지, 포트, 생성일, 메타태그 아이콘, 진행률 바

### 3. Ideas 전용 페이지
- 1_idea_stage 프로젝트만 표시 (칸반에서 분리)
- Promote to Initiation (작업지시 모달 연동)
- Discard → 7_discarded 이동

### 4. 프로젝트 상세
- label / description 인라인 편집 (클릭)
- Documents 탭: 파일 목록 + MD 에디터 (분할 뷰), 새 파일 생성, 다중 선택 삭제
- Instructions 탭: 수동 작업지시 생성 (커스텀 체크리스트)
- Settings 탭: 메타태그 편집, Timeline & Progress, Related People

### 5. 메타태그 시스템
- YAML frontmatter 기반 (blockquote에서 마이그레이션 완료)
- 중요도 (★1-5), 위급도 (low~critical), 긴급도 (low~urgent)
- 협업 (personal/collaboration), 주도 (lead/member), 오너
- 유형 (개발/연구/연구+개발), 포트
- 목표종료일, 실제종료일
- 서브태스크 (total/done) + 진행률 바 (4등분 마커)
- Related People (People 메뉴 연동 태그)

### 6. 문서 관리
- 프로젝트별 docs/ 브라우저
- 공통 폴더: Notes, Learning, Issues, Guidelines
- MD 에디터 (@uiw/react-md-editor) — 편집/프리뷰 분할 뷰
- 파일 생성, 단일 삭제, 다중 선택 삭제
- 폴더 생성/삭제 API
- Description 수정 시 _아이디어노트.md + README.md + _토의록.md 동기화

### 7. Quick Note
- 사이드바에서 어디서나 열 수 있는 슬라이드 패널
- _notes/_temp/에 저장, 5개 카테고리로 이동/정리

### 8. 작업지시 시스템
- 단계 전환 시 자동 생성 (작업지시_YYYY-MM-DD.md)
- 프로젝트 내 수동 생성 (Instructions 탭)
- 같은 날 여러 지시 → 시간 스탬프로 append
- 기본 체크리스트 포함 ("Update work instruction status upon completion")
- Claude Code 프롬프트에 체크리스트 업데이트 지시 자동 포함

### 9. Work Execution
- 사이드바 Execute → 미완료 작업지시 스캔
- Launch Claude Code → 임베디드 터미널 (xterm.js + WebSocket PTY)
- 세션 종료 시 목록 자동 새로고침

### 10. Work Status
- 사이드바 Status → 전체 프로젝트 작업현황 대시보드
- 프로젝트별 진행률, 체크리스트 상세

### 11. People
- 관계자 카드 프로파일 (이름, 소속, 역할, 전문분야, 관계)
- Connections (연관 인물 네트워크)
- 프로젝트 연결
- 기본 데이터: Chad (Chungil Chae)

### 12. Trash
- 7_discarded 프로젝트 리스트
- 복원 (→ 1_idea_stage) / 영구 삭제 (폴더 + 앱 양쪽)

### 13. 서버 제어
- 프로젝트별 start/stop/restart (run.sh 연동)
- 포트 스캔, 서버 상태 모니터링
