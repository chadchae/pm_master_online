# PM Master Online

Projects 폴더 내 모든 프로젝트, 문서, 노트, 이슈, 학습, 가이드라인을 웹에서 운영·관리하는 로컬 프로젝트 매니저. V1과 기능 동일, 활성 개발 버전.

---

## 구현 완료 기능 (2026-03-20)

### Core
1. 비밀번호 인증 (bcrypt, 파일 기반 토큰)
2. 칸반/리스트 대시보드 + 카드 순서 드래그
3. 칸반 sublabel (연구 맥락: Discussion / Data & Lit Review / Analysis / Writing / Submitted)
4. Ideas 전용 페이지 + 새 아이디어/프로젝트 생성
5. 프로젝트 상세 (label/desc 인라인 편집, 5탭)
6. 메타태그 (중요도/위급도/긴급도/유형/포트/협업/오너)
7. YAML frontmatter 표준

### Project Tabs
8. Documents: MD 에디터, 다중 삭제, 폴더 drill-down, 새 파일/폴더
9. Instructions: 수동 작업지시 + 전환 자동 생성
10. Todo 칸반 (체크박스/담당자/마감일/완료일)
11. Issue Tracker (스레드/댓글 CRUD/상태/라벨/필터 카운트)
12. Settings: 메타태그, Timeline & Progress, 서브태스크 (CRUD/드래그/Done/Cancel), Related People
13. Schedule: 테이블 뷰 + Gantt 차트 + 마일스톤

### Schedule System (NEW)
14. Schedule 테이블 뷰: 카테고리, 의존성, 타입, 담당자, 날짜, 기간
15. Gantt 차트: 날짜 기반 바, today line, auto-scroll, 카테고리별 그룹
16. 마일스톤 지원 (다이아몬드 아이콘, gantt 표시)
17. 의존성 자동 날짜 계산 (선행 작업 종료일 → 후행 작업 시작일)
18. 타입 필터 (task/milestone/review/meeting + custom)
19. 카테고리/부모 작업 연동
20. Schedule summary 위젯 (대시보드, 실시간 데이터)
21. 기간 계산 inclusive (시작일·종료일 모두 포함)

### UI/UX Improvements (NEW)
22. AppDialogs: 브라우저 prompt/confirm → 커스텀 인앱 모달
23. 문서 Print/PDF 버튼 (window.open + window.print)
24. 필터 상태 localStorage 영속화
25. 대시보드 필터/소팅 개선
26. Markdown lazy-load 렌더링 (@uiw/react-markdown-preview)

### Global Features
27. Quick Note (_temp/ 연동)
28. Work Execution (임베디드 터미널 xterm.js + WebSocket PTY)
29. Work Status 대시보드
30. People (카드/connections/Related People 자동생성)
31. Trash (복원/영구삭제)
32. 서버 제어 + 로그 뷰어
33. 토의록 타임라인
34. 프로젝트 다운로드 (zip)
35. 카드 액션 (수정/삭제/다운로드)
36. i18n (한/영, 280+ 키)
37. 컬럼 소트 (프로젝트 리스트)
